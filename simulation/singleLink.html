SINGLE LINKAGE

Single linkage clustering defines the distance between two clusters as the minimum distance between any pair of points belonging to different clusters.

How it works:

1. Each data point is initially treated as a separate cluster.
2. All pairwise distances between data points are computed.
3. The two closest points (having minimum distance) are identified and merged together to form a single cluster.
4. Distances between clusters are recalculated using the single linkage rule, where the minimum inter-point distance is considered.
5. The process is repeated until the stopping condition is satisfied or a single cluster remains.
